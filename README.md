# Biirr 
App that present a list of beers from the BreweryDB APIs.

## Endpoint:
https://api.brewerydb.com/v2/beers

Please get your own API key at https://www.brewerydb.com

## Features: 
- UITableView 
- UIScollView 
- JSON decoding 

## Dependencies:
SDWebImage to read and cache images from URL
